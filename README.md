# skipedia

## サイト概要

スキーの専門用語をだれでも投稿閲覧できるサイト

### サイトテーマ

スキー専門用語投稿サイト

### テーマを選んだ理由

スキーの専門用語専用投稿サイトが世の中に存在しないから

### ターゲットユーザ

スキーヤー（特に基礎スキーヤー）

### 主な利用シーン

・練習中や言葉の意味がわからなかった時パッと調べられる
・参考になった表現や動画、記事を誰でも公開できる
